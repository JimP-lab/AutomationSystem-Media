## AutomationSystem-Media ## 

AutomationSystem-Media is an automatic system that posts every hour posts in social media using switchboard for creating image posts and social bee for uploading and managing and by using the graph api of meta for bringing the latest posts that were uploaded and it is matching the corresponding link to corresponding post and it using coding to extract the content from rss feeds by managing multiple websites at the same time with the business manager and it was created with the help of social ai automation assistant.
